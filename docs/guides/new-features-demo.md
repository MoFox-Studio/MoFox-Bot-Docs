# ✨ 新功能演示

本文档用于演示 MoFox Bot 文档站点的新增功能。

## 📊 阅读进度条

请观察页面顶部。当你向下滚动此页面时，顶部的渐变色进度条会随之增长。

::: tip 提示
进度条在移动端会自动变细，以免遮挡内容。
:::

## 📋 代码块复制增强

将鼠标悬停在下方的代码块上，你会看到右上角出现一个复制按钮。点击它即可复制代码。

```javascript
// 这是一个示例代码块
function hello() {
  console.log("Hello, MoFox!");
}
```

```python
# Python 代码示例
def greet(name):
    return f"Hello, {name}!"
```

## ⌨️ 全局快捷键

你可以随时使用键盘快捷键来导航文档：

1. 按 **`?`** (Shift + /) 打开快捷键帮助弹窗。
2. 按 **`t`** 快速回到页面顶部。
3. 按 **`b`** 快速滚动到底部。
4. 按 **`Esc`** 关闭帮助弹窗。
